# ロバのみみ

ここは「バーチャル王様の耳はロバの耳」（ロバ耳）プロジェクトです。

## なにこれ？

言いたいけど言えない、誰にも話せないこと、ありませんか？

「ロバ耳」は、声にならない声を受け取る場所です。
投げ銭（有料）で相談も受け付けています。

- チケット購入ページ（準備中）

## なぜ GitHub？

「ロバ耳」は、広告やSNSの喧騒から離れた場所にひっそり開かれた穴。  
GitHub Pages なら、それができる気がしたからです。
